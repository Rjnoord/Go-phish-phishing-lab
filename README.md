# GoPhish Phishing Simulation Lab

## Overview
This lab demonstrates the deployment and use of the GoPhish framework to conduct a simulated phishing campaign. The goal is to understand how phishing attacks are created, delivered, and measured, and how organizations can assess user security awareness.

## What I Did
- Installed and configured the GoPhish phishing framework
- Created phishing email templates
- Built a credential capture landing page
- Launched a simulated phishing campaign
- Monitored user interaction and campaign metrics
- Analyzed phishing effectiveness and risk

## Skills Demonstrated
- Phishing awareness & analysis
- Blue team / defensive security concepts
- Security risk assessment
- Linux system administration
- Ethical use of offensive security tools

## Tools & Technologies
- GoPhish
- Linux
- SMTP / Email concepts
- Web-based security dashboards
